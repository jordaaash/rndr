# RNDR example
An example of using the RNDR client library to create a simple service that handles requests to fund jobs and disburse funds.

## Run

```shell
git clone https://github.com/jordansexton/rndr.git

cd rndr/js
yarn install
yarn build

cd example
yarn install
yarn start
open http://localhost:3000
```
